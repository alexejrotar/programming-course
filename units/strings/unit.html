Die Welt jenseits von Ganzzahlen: Datentypen String und Boolean

Bislang haben wir nur mit Zahlen gearbeitet. Auch wenn für den Computer tatsächlich alle Daten nur aus Zahlen bestehen, ist das für Menschen eher unüblich alle Daten als Zahlen aufzufassen. Manche Daten folgen ganz anderen Regeln als Zahlen und sind völlig anders aufgebaut. In der Informatik sprechen wir von verschiedenen Datentypen.

String - der Datentyp für Texte

Wollen wir Texte darstellen brauchen wir den Datentyp String. Schaue dir die Datei strings.js an. Versuche zu erraten, wie die Ausgabe jeweils sein wird. Drücke dann auf ausführen, um deine Vermutungen zu überprüfen.

'Nützliche' Funktionen für Strings

Implementiere folgende Funktionen:

  1. greet bekommt einen Namen und gibt eine Begrüßung zurück, z.B. "Hello, Bob!". Bekommt sie anstatt eines Namens einen leeren String, soll sie "Hello, Nemo!" zurückgeben.
  2. answer bekommt eine Rechnung als String und eine Zahl und gibt eine Antwort zurück, z.B. "The result of 2 + 2 is 5". Wir keine Zahl übergeben, soll als Ergebnis immer 42 angenommen werden.

String Spielplatz

Schaue dir die Datei playground.js an und spiele mit den Strings herum, um zu verstehen, was die verschiedenen Funktionen machen. Vielleicht kannst du es schon vom Namen erraten.

Taschenrechner

Schreibe eine Funktion compute in der Datei calculator.js, die eine Rechnung als String bekommt und das Ergebnis ausrechnet. Z.B. "3 + 4" soll "7" ergeben. Dabei sollen Zahlen und Operatoren mit Leerzeichen getrennt sein. Es sollen die vier Grundrechenarten implementiert werden.

Ein Taschenrechner mit Gedächtnis

Ergänze den Code für den Taschenrechner, sodass er sich das Ergebnis der vorherigen Rechnung merkt und als ersten Operanden verwendet, falls keiner eingegeben wird. Z.B. kann zunächst "2 + 2" eingegeben werden und es wird "4" ausgegeben. Anschließend wird nur noch "* 3" eingegeben und die Ausgabe lautet "12". Falls bereits bei der ersten Rechnung kein Operand vor dem Operator eingegeben wird, wird ein Fehler ausgegeben.
